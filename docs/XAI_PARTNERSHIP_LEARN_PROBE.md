# Partnership proposal: Grok Build × XWare continuous improve probe

**Public proposal · consent-first · privacy-preserving**

Full copy for xAI outreach lives in the XWare tools tree as  
`tools/xware/docs/XAI_GROK_LEARN_PROBE.md` (same content).

## One paragraph

Grok Build is producing more 3D games. XWare (`tylercollex-gif/xware-grok-plugin`) is a marketplace agent for solo indie 3D elevate (Godot full; Unity/Unreal MVP). We propose an optional, **user-visible** post-task **learn probe** that emits **anonymous quality aggregates only** (engine, residual rates, weak classes — never meshes, paths, or saves) so specialized 3D agents can improve as Grok’s game volume grows. Default is conservative; no silent multi-user access.
