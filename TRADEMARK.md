# XWare trademark & identity policy

**Goal:** XWare stays **free and available for everyone**, while the **name,
agent identity, and official listing** stay with the creator.



## Reserved marks

Without prior written permission from **tylercollex-gif**, do not use:

| Mark | Scope |
|------|--------|
| **XWare** | Product name, agent name, plugin id, marketplace listing title |
| **Xhance** | Product mode / skill family name |
| Official logos / wordmarks | Any branding that implies the same product |

in a way that would confuse people into thinking a fork, remake, or
rebrand **is** the official XWare project or agent.

## Free use that is always OK

Under the MIT License you **may**:

- Use XWare tools and the agent **for free** in personal and commercial games
- Fork the repo, study it, fix bugs, send pull requests
- Say your project is “built with XWare” or “compatible with XWare” (truthful)
- Quote short names in academic or news coverage

## Not OK (identity theft / remake-as-official)

You **may not**:

- Publish a marketplace plugin, agent, or skill pack named **xware** / **XWare**
  that is a rebranded remake claiming original authorship
- Use `subagent_type=xware` branding, slash commands `/xware*`, or the same
  agent card text to present a different product as the official agent
- Remove copyright / NOTICE and claim sole original authorship of substantial
  XWare work
- Imply xAI or tylercollex-gif endorsement without permission

## Allowed forks (honest)

Forks should:

1. Keep MIT copyright + NOTICE
2. Use a **different product name** (not XWare / Xhance as the primary brand)
3. State clearly: “Derived from XWare by tylercollex-gif” with a link to the
   official repo
4. Not take over the marketplace name `xware` or official pin

## Official agent only

| Field | Official value |
|-------|----------------|
| Creator | **tylercollex-gif** |
