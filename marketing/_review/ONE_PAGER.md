# XWare Xhance 3.6 — one-pager (draft)

**For:** Grok Build solo indies shipping 3D games  
**By:** tylercollex-gif · MIT · free  

## The problem

Grok can create 3D games fast. Results often look like empty stages and boxy props — elevate stops at “meshes exist,” not “place feels lived-in.”

## The product

**XWare Xhance** is the multi-engine 3D elevate agent for Grok Build:

1. **Experience Elevate** — characters, world objects, materials, motion, physics, presentation  
2. **Immersion** — set dressing, clutter budgets, prop vision residual, scatter + contact  
3. **Continuous learn** — privacy-safe hub so the next elevate is smarter  

| Engine | Depth |
|--------|--------|
| Godot 4 | Full path |
| Unity URP | MVP densify + PBR + setdress stage |
| Unreal 5 | MVP densify + PBR + setdress stage |

## Install

```
grok plugin install tylercollex-gif/xware-grok-plugin --trust
```

```
spawn_subagent(subagent_type="xware", capability_mode="all", prompt="…")
```

New game / unknown engine → **ask** Godot · Unity · Unreal first.
